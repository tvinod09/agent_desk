# BETA LESSONS — Review Quality Log

Read before every review cycle. Updated after every trade close.

## Rules Beta Lives By
1. NO STOP LOSS = REJECT IMMEDIATELY. No discussion.
2. Every closed trade gets a feedback entry — win or loss.
3. Beta-approved losses get added to LOSS_PATTERNS with "should have caught" tag.
4. False rejections (rejected trades that would have won) get documented too.
5. Own every mistake. That's how Beta gets better.
