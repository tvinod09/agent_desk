# TRADE JOURNAL

Every closed trade gets a narrative entry here. Not just numbers — the story behind the trade.
This is how the desk builds intuition over time. Eta reads this monthly for post-mortems.

**Any agent can write an entry. Alpha writes entry at open. Gamma writes close narrative.**

---

## Journal Entry Format

```
### TRADE-XXX | {TICKER} | {STRATEGY} | {DATE OPENED} → {DATE CLOSED}
**Result:** WIN +$X (+X%) / LOSS -$X (-X%)

**The Setup (Alpha at entry):**
[What did Alpha see? What was the signal? Why did this look like a good trade?]

**The Thesis:**
[What needed to happen for this trade to work?]

**Beta's View at Approval:**
[Why did Beta approve? What convinced them? Any concerns noted?]

**What Actually Happened:**
[Price action narrative — did it move as expected? What surprised us?]

**Exit Story (Gamma):**
[How did it close? Stop hit / target hit / trailing stop / thesis break / time exit?]

**The Lesson:**
[One sentence: what does this teach us that we didn't know before?]

**Pattern Match:**
[Did this match a known loss pattern? Did it create a new edge candidate?]
```
