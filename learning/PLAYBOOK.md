# PLAYBOOK.md — The Desk's Documented Edges

This file is the collective intelligence of the desk. Every time an agent discovers a repeatable setup that works, it gets documented here. This is NOT a strategy list — it's specific, tested plays with real data behind them.

**All agents read this before every cycle. Alpha uses it to prioritize setups. Beta uses it to calibrate approval confidence. Eta updates it weekly.**

---

## How a Play Gets Added
1. A setup wins 3+ times with similar conditions
2. Alpha or any agent nominates it with evidence
3. Beta validates the pattern
4. Eta confirms with historical data
5. Play added with confidence rating

## Confidence Ratings
- ⭐⭐⭐⭐⭐ (90%+ win rate, 10+ occurrences) — trade full size
- ⭐⭐⭐⭐ (75%+ win rate, 7+ occurrences) — trade normal size
- ⭐⭐⭐ (60%+ win rate, 5+ occurrences) — trade 50% size
- ⭐⭐ (50%+ win rate, 3+ occurrences) — small size, monitor
- ⭐ (unconfirmed, <3 occurrences) — paper track only
